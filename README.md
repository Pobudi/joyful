# medical-outfits-store

This project is a static website for an online store selling medical outfits. It showcases various products and provides information about the store.

## Project Structure

```
medical-outfits-store
├── src
│   ├── css
│   │   ├── main.css
│   │   └── normalize.css
│   ├── js
│   │   └── main.js
│   └── pages
│       ├── about.html
│       ├── catalog.html
│       ├── contact.html
│       └── index.html
├── assets
│   └── styles
│       └── fonts.css
├── package.json
└── README.md
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd medical-outfits-store
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open the `src/pages/index.html` file in your browser to view the website.

## Features

- Responsive design for optimal viewing on various devices.
- A catalog of medical outfits available for purchase.
- Contact form for inquiries.
- About page detailing the store's mission and values.
